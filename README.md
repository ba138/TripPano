# 🏨 TripPano

TripPano is a modern **hostel booking iOS application** built with **SwiftUI**. It provides a clean, intuitive, and responsive experience for travelers to discover, explore, and book hostels with ease.

## ✨ Features

- 🔐 User Authentication
- 🏨 Browse and search hostels
- 📍 View hostel details
- 🖼️ High-quality hostel image galleries
- ❤️ Save favorite hostels
- 📅 Easy hostel booking flow
- 🔍 Search and filtering
- 👤 User profile management
- 📱 Responsive SwiftUI interface
- 🌙 Clean and modern UI/UX

## 🛠️ Tech Stack

- Swift
- SwiftUI
- MVVM Architecture
- REST APIs
- URLSession
- Async/Await
- Codable
- Git & GitHub

## 🚀 Getting Started

### Prerequisites

- Xcode 16+
- iOS 17+
- macOS Sonoma or later

### Installation

1. Clone the repository

```bash
git clone https://github.com/ba138/TripPano.git
```

2. Open the project

```bash
cd TripPano
open TripPano.xcodeproj
```

3. Build and run the project using Xcode.
